# Data Notice

HoneyPrint(TM) is GoldenFlow Labs' proprietary spectral database and is not included in this public repository.

## HoneyPrint Overview

- Contains 1,000+ labeled honey samples
- Covers 5 adulterant risk profiles used in internal research workflows
- Includes curated metadata and quality-controlled spectral records

## Why It Is Not Public

HoneyPrint is excluded from this repository to protect intellectual property and maintain controlled research-grade data governance.
